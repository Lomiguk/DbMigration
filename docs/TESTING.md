## Уровни тестирования

| Уровень | Тип | Расположение | Время | Зависимости |
|---------|-----|--------------|-------|-------------|
| **Unit** | Отдельные компоненты | `src/test/kotlin/unit/` | < 5 сек | Нет |
| **Integration** | С реальной БД | `src/test/kotlin/integration/` | 1-2 мин | Docker |
| **Benchmark** | Сравнение производительности | `src/test/kotlin/benchmark/` | 2-5 мин | Docker |

### Библиотеки

| Библиотека | Версия | Назначение |
|------------|--------|------------|
| JUnit 5 | 5.10.0 | Фреймворк |
| AssertJ | 3.24.2 | Fluent assertions |
| Testcontainers | 1.19.3 | PostgreSQL контейнеры |
| MockK | 1.13.8 | Мокирование |

## Запуск тестов

```bash
# Все тесты
./gradlew test

# Только unit
./gradlew test --tests "unit.*"

# Только integration (требуется Docker)
./gradlew test --tests "integration.*"

# Только benchmark (требуется Docker)
./gradlew test --tests "benchmark.*"

# Конкретный тест
./gradlew test --tests "unit.DependencyResolverTest"

# С логами
./gradlew test --info
```

## Интеграционные тесты

### MappingServiceTest (15 тестов)

- Создание таблицы `migration_mapping`
- Сохранение в кэш и БД
- Пакетная обработка (batch)
- Получение всех UUID таблицы
- Ограничение размера кэша

### MetadataReaderTest (10 тестов)

- Чтение таблиц с UUID PK
- Чтение внешних ключей
- Чтение колонок
- Определение типов данных

### DataMigratorTest (12 тестов)

- Создание целевой схемы (BIGINT)
- Миграция данных с конвертацией FK
- Пропуск уже мигрированных записей
- Генерация уникальных BIGINT ID
- Инкрементальная миграция

### DataIntegrityTest (8 тестов)

- Сохранение ссылочной целостности FK
- Полнота данных (без потерь)
- Консистентность маппинга (1:1)
- Каскадное удаление
- Пост-синхронизация валидация

### ChangeCaptureTest (10 тестов)

- Синхронизация только новых данных
- Пропуск мигрированных записей
- Memory-Filtered стратегия (HashSet)
- Производительность на больших дельтах
- Непрерывная репликация (multiple rounds)

## Troubleshooting

### "Could not find a valid Docker environment"

```bash
docker ps
docker compose up -d
```

### "Connection timeout"

Увеличить timeout в `BaseIntegrationTest.kt`:
```kotlin
connectionTimeout = 30000
```

### "OutOfMemoryError"

```kotlin
// build.gradle.kts
jvmArgs = listOf("-Xmx4G", "-XX:+UseG1GC")
```

### Тесты слишком долгие

```bash
# Запустить только unit
./gradlew test --tests "unit.*"

# Очистить после неудачных
./gradlew cleanTest
docker container prune -f
```
